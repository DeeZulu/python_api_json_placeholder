# python_api_json_placeholder

Pet проект по автоматизации тестирования API: https://jsonplaceholder.typicode.com

Cоздать и активировать виртуальное окружение (Mac)
`python3 -m venv venv`
`source bin/venv/activate`

Далее выбрать интерпретатор из созданного окружения

Установить зависимости
`pip install -r requirements.txt`
